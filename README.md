# My Portfolio Website

A clean, modern portfolio website built with plain HTML, CSS, and JavaScript.

## 📁 Project Structure

```
portfolio/
├── index.html      # Main HTML file with page structure
├── styles.css      # All styling and responsive design
├── script.js       # JavaScript for interactivity
└── README.md       # This file
```

## 🚀 Getting Started

### 1. **View Your Website**
   - Open `index.html` in your browser
   - Or use VS Code's Live Server extension for auto-refresh

### 2. **Customize Your Portfolio**

   **In `index.html`:**
   - Replace "Your Name" with your actual name
   - Update the tagline
   - Add your projects with descriptions
   - Add your skills
   - Update contact links (email, GitHub, LinkedIn, etc.)

   **In `styles.css`:**
   - Change colors by editing the `:root` variables at the top
   - Customize fonts, spacing, and sizes as needed
   - Add your own animations or effects

   **In `script.js`:**
   - Add more interactivity as you learn JavaScript
   - Uncomment the dark mode code if you want that feature

## 🎨 Color Scheme

The current colors use:
- **Primary Blue**: `#3498db`
- **Dark Gray**: `#2c3e50`
- **Light Gray**: `#ecf0f1`

Change these in the `:root` section of `styles.css`

## 🔧 Common Customizations

### Add a Profile Image
```html
<img src="images/profile.jpg" alt="My photo" class="profile-pic">
```

### Change Colors
Edit in `styles.css`:
```css
:root {
    --primary-color: #your-color;
    --secondary-color: #your-color;
}
```

### Add More Sections
Copy a section block in `index.html` and add it to the navigation
